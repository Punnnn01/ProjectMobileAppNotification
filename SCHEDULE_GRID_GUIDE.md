# 📅 Schedule Grid Feature

## 🎯 ฟีเจอร์ที่เพิ่ม

### **1. ตารางเรียน/ตารางสอนแบบ Grid**
- แกน X: วันจันทร์ - อาทิตย์ (7 วัน)
- แกน Y: เวลา 8:00 - 20:00 (13 ช่วงเวลา)
- รองรับ **นิสิต** (จัดตารางเรียน) และ **อาจารย์** (จัดตารางสอน)

### **2. การจัดการวิชา**
- แตะช่องว่างเพื่อเพิ่มวิชา
- กำหนดระยะเวลา 1-3 ชั่วโมง
- กดค้างเพื่อลบวิชา
- เช็คตารางซ้อนทับอัตโนมัติ

### **3. ข้อมูลวิชา**
- รหัสวิชา (เช่น 01418113)
- ชื่อวิชา
- ห้องเรียน (Optional)
- สีสุ่มอัตโนมัติ

---

## 🗄️ Database Structure

### **Student Collection:**
```javascript
Student/{student_id}
{
  student_id: String,
  personal_info: Map,
  role: Map,
  subject: Array,
  notification: Array,
  bookmarks: Array,
  
  // ⭐ เพิ่มใหม่:
  schedule: Array<ScheduleItem>
}
```

### **Teacher Collection:**
```javascript
Teacher/{teacher_id}
{
  teacher_id: String,
  personal_info: Map,
  role: Map,
  subject: Array,
  notification: Array,
  bookmarks: Array,
  
  // ⭐ เพิ่มใหม่:
  schedule: Array<ScheduleItem>
}
```

### **ScheduleItem Structure:**
```javascript
{
  id: String,                  // Unique ID
  day: Number,                 // 0-6 (จันทร์-อาทิตย์)
  startHour: Number,           // 8-20
  endHour: Number,             // 9-21
  subjectCode: String,         // รหัสวิชา
  subjectName: String,         // ชื่อวิชา
  room: String,                // ห้องเรียน (optional)
  color: String                // สี hex (เช่น "#3B82F6")
}
```

**ตัวอย่าง:**
```javascript
{
  id: "1701234567890",
  day: 1,                      // อังคาร
  startHour: 9,                // 9:00
  endHour: 11,                 // 11:00
  subjectCode: "01418113",
  subjectName: "Computer Programming",
  room: "EN-205",
  color: "#3B82F6"
}
```

### **Step 2: ทดสอบบน Mobile App**

1. **เปิดแอป → Login**
2. **ไปหน้า Schedule**
3. **แตะช่องว่าง** (เช่น จันทร์ 9:00)
4. **กรอกข้อมูล:**
   - รหัสวิชา: `01418113`

   - ห้องเรียน: `EN-205`
   - ระยะเวลา: `2 ชม.`
5. **กด "เพิ่มวิชา"**
6. **วิชาจะปรากฏในตาราง!**

## 🔄 Flow การทำงาน

### **1. เพิ่มวิชา:**
```
User แตะช่องว่าง (เช่น จันทร์ 9:00)
    ↓
เปิด Modal
    ↓
User กรอกข้อมูล (รหัส, ชื่อ, ห้อง, ระยะเวลา)
    ↓
กด "เพิ่มวิชา"
    ↓
เช็คตารางซ้อนทับ
    ↓
ถ้าไม่ซ้อน → บันทึกลง Firestore
    ↓
อัปเดตตาราง
```

### **2. ลบวิชา:**
```
User กดค้างที่วิชา
    ↓
แสดง Alert ยืนยัน
    ↓
กด "ลบ"
    ↓
ลบจาก Firestore
    ↓
อัปเดตตาราง
```
